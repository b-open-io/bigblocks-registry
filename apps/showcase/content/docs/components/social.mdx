## Key Features

- **Decentralized Identity**: Built on Bitcoin-based user profiles
- **Privacy-Preserving**: Users control their social graph
- **Interoperable**: Works across different platforms
- **Real-time**: WebSocket support for live updates
- **Customizable**: Adapt to your brand and design
