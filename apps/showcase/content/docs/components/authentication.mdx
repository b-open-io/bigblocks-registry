## Overview

BigBlocks provides a comprehensive suite of authentication components built on Bitcoin's cryptographic foundation. These components enable passwordless authentication using Bitcoin key pairs.

## Key Features

- **Passwordless**: No passwords to remember or manage
- **Self-sovereign**: Users control their own identity
- **Secure**: Built on Bitcoin's proven cryptography
- **Privacy-focused**: Minimal data collection
- **Developer-friendly**: Easy integration with existing apps
